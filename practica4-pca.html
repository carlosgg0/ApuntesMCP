Ejercicio 5

Ahora realizamos el PCA sobre los datos estandarizados:

pca_wine <- prcomp(wine_data, scale = TRUE)

Para ver cuánta varianza explican las dos primeras componentes podemos acceder al elemento pca_wine$sdev, ya que pca_wine$sdev^2 representa la varianza explicada por cada componente:

pca_wine$sdev^2
 [1] 4.7058503 2.4969737 1.4460720 0.9189739 0.8532282 0.6416570 0.5510283
 [8] 0.3484974 0.2888799 0.2509025 0.2257886 0.1687702 0.1033779

Para entenderlo mejor podemos obtener la proporción de varianza explicada:

prop_varianza <- pca_wine$sdev^2 / sum(pca_wine$sdev^2)

En este caso son V7, V6 y V12 (en ese orden).

Esto nos dice que, la primera componente del análisis intenta medir mayoritariamente la cantidad de dichas variables:

  • V7 (Flavanoids)
  • V6 (Total phenols)
  • V12 (OD280/OD315 of diluted wines)
