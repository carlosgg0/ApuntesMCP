4  Autovalores, Autovectores y Descomposición Singular

4.1 Breve recordatorio

La definición de autovalor \(\lambda\) y autovector \(v\) es la siguiente:

\[ Av = \lambda v \]

Intuitivamente se puede entender como los espacios vectoriales cuya dirección no se ve afectada cuando se les aplica la transformación lineal \(A\).

Desarrollando la ecuación:

\[ (A-\lambda I)v = 0 \]

por lo que es necesario que \(det(A-\lambda I) = 0\) para que existan soluciones. Ecuación con la que obtenemos \(\lambda\).

4.2 El método de la potencia

Dado un vector inicial aleatorio \(b_0\):

\[ b_{k+1} = \frac{Ab_k}{\|Ab_k\|} \]

Un ejemplo de ejecución se vería tal que así:

\[ b_0 \implies b_1=Ab_o \implies b_2=Ab_1 \implies \ldots \implies b_{k+1}=Ab_{k} \]

(dividiendo por la correspondiente norma).

Esta serie de productos siempre converge al autovalor de mayor valor absoluto, denomidado autovalor dominante.

Posible función que calcula el método de la potencia:

power_method <- function(A, epsilon = 1e-6, max_iter = 1000) {
  # 1. Vector inicial aleatorio (normalizado)
  n <- nrow(A)
  x <- rnorm(n)
  x <- x / sqrt(sum(x^2))
  
  lambda_old <- 0
  
  for(i in 1:max_iter) {
    # 2. Multiplicación (aprovecha la dispersión de A si es sparse)
    x_new <- as.vector(A %*% x)
    
    # 3. Estimación del autovalor (Cociente de Rayleigh simple)
    # Si x está normalizado, lambda es aproximadamente t(x) %*% A %*% x
    # Aquí simplificamos usando la norma del nuevo vector antes de normalizar
    lambda <- sqrt(sum(x_new^2)) 
    
    # Recuperar signo si es necesario (para autovalores reales)
    # if(sum(x_new * x) < 0) lambda <- -lambda
    
    # 4. Normalización
    x_new <- x_new / sqrt(sum(x_new^2))
    
    # 5. Convergencia
    if(abs(lambda - lambda_old) < epsilon) {
      return(list(
        eigenvalue = lambda, 
        eigenvector = x_new, 
        iterations = i
      ))
    }
    
    x <- x_new
    lambda_old <- lambda
  }
  warning("El método no convergió")
  return(list(eigenvalue = lambda, eigenvector = x, iterations = max_iter))
}

4.3 El algoritmo QR

Mientras que el método de las potencias encuentra solo el autovalor dominante, el algoritmo QR encuentra todos los autovalores de una matriz (el espectro completo).

qr_algo <- function(A, max_iter = 20, epsilon = 1e-6) {
  A_k <- A
  # Guardamos la evolución para visualizar (opcional)
  historia <- numeric(max_iter)
  
  for(i in 1:max_iter) {
    # 1. Descomposición QR: A = Q * R
    descomp <- qr(A_k)
    Q <- qr.Q(descomp)
    R <- qr.R(descomp)
    
    # 2. Actualización: A_new = R * Q
    A_new <- R %*% Q
    
    # Guardamos el valor de la esquina superior (para ver convergencia)
    historia[i] <- A_new[1,1]
    
    # Cuando la diferencia sea despreciable nos salimos
    if (max(abs(A_new - A_k)) < epsilon) {
      break
    }
    
    A_k = A_new
  }
  
  return(list(matriz_final = A_k, 
              autovalores = diag(A_k),
              iteraciones = i))
}

4.3.1 Consideraciones

Si necesitamos el autovalor dominante es recomendable usar el método de la potencia definido por nosotros; si necesitamos los \(k\) primeros autovalores, usamos el paquete rspectra; y por último, si los queremos todos usamos el método eigen del R base.

4.3.2 Ejemplo de Dinámica de poblaciones

# Matriz de Leslie L:
# Juveniles -> Pre-adultos -> Adultos
L <- matrix(c(0,   0.5, 2.0,  # Fecundidad (solo adultos producen mucho)
              0.1, 0,   0,    # 10% de juveniles sobrevive a pre-adultos
              0,   0.5, 0),   # 50% de pre-adultos sobrevive a adultos
            nrow = 3, byrow = TRUE)

# Autovalores
eigen_L <- eigen(L)
lambda_dominante <- Re(eigen_L$values[1])

cat("Tasa de crecimiento a largo plazo (lambda):", round(lambda_dominante, 3), "\n")
