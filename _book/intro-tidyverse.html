2.1 Pipes

Un pipe es un operador que permite encadenar varias funciones, de forma que el resultado de una función se pasa como argumento a la siguiente función.

Son ampliamente usados en el ecosistema tidyverse, aunque no son exclusivos de este.

log(exp(sin(cos(abs(1:10)))))
 [1]  0.5143953 -0.4042392 -0.8360219 -0.6080830  0.2798734  0.8192892
 [7]  0.6844888 -0.1449872 -0.7901969 -0.7440231

Es equivalente a:

1:10 |>
  abs() |>
  cos() |>
  sin() |>
  exp() |>
  log()
 [1]  0.5143953 -0.4042392 -0.8360219 -0.6080830  0.2798734  0.8192892
 [7]  0.6844888 -0.1449872 -0.7901969 -0.7440231

La librería se puede instalar mediante el comando install.packages("tidyverse") en la consola.

library(tidyverse)
data(starwars)

2.6 La función summarise()

Para resúmenes estadísticos descriptivos.

Ejemplo: peso y estatura medias en el dataset:

# Con R base
mean(starwars$mass, na.rm = TRUE)
[1] 97.31186
mean(starwars$height, na.rm = TRUE)
[1] 174.6049
# Con tidyverse
starwars |>
  summarise(
    peso_medio = mean(mass, na.rm = TRUE),
    altura_media = mean(height, na.rm = TRUE)
  )
# A tibble: 1 × 2
  peso_medio altura_media
       <dbl>        <dbl>
1       97.3         175.

Ejemplo: peso y estatura medias en el dataset, por cada especie:

starwars |>
  group_by(species) |>
  summarise(
    peso_medio = mean(mass, na.rm = TRUE),
    altura_media = mean(height, na.rm = TRUE),
    numero_individuos = n() # número de filas de cada grupo
  )
